# 行业研究 -- 行业全景研究报告

撰写行业全景研究报告，覆盖市场空间测算、产业链图谱、竞争格局、驱动因素分析、政策梳理和投资机会。

**重要提示**：本技能辅助行业研究工作流程，不构成投资建议。市场空间测算和投资机会梳理涉及多项假设，仅供研究参考。

## 输入要求

确认：
1. **行业名称**：申万一级/二级分类或具体赛道
2. **研究深度**：全景概览/重点环节深挖
3. **侧重方向**：市场空间/竞争格局/政策梳理/产业链/投资机会
4. **可用材料**：行业白皮书/政策文件/上市公司年报——材料越充分质量越高

## 执行流程

### 第一步：行业定义与分类

明确行业边界：行业定义、与相邻行业的区分、申万行业分类位置、产业链上中下游拆分。

**行业生命周期判断**：

| 阶段 | 特征 | 投资关注点 |
|------|------|-----------|
| **导入期** | 渗透率<5%，高增速但规模小，技术路线未定 | 技术路线选择、先发优势 |
| **成长期** | 渗透率5%-30%，快速放量，竞争格局初步形成 | 市占率提升、规模效应拐点 |
| **成熟期** | 渗透率>30%，增速放缓，格局稳定 | 盈利能力、分红、效率提升 |
| **衰退/转型期** | 增速<GDP增速，面临替代威胁 | 转型路径、剩余价值 |

### 第二步：市场空间测算

> 市场空间测算依赖渗透率、单价、增速等核心假设，不同假设下结论可能差异较大。

**自上而下法**（Top-down）：
- 全球市场 = [基数] x [渗透率] x [单价]
- 中国市场 = 全球市场 x [中国份额] 或 [中国特有基数] x [渗透率] x [单价]
- 增速 = [量增] x [价增] + [新应用场景增量]

**自下而上法**（Bottom-up）：
- 按终端应用分拆：应用A需求量 x 单价 + 应用B需求量 x 单价 + ...
- 按产能/出货量推算：主要厂商产能合计 x 产能利用率 x ASP

**交叉验证**：两种方法结果偏差应<20%，偏差过大须解释原因。

核心假设确认点：渗透率/单价/增速假设是否合理？

### 第三步：产业链分析

梳理产业链图谱：

| 环节 | 核心内容 | 分析要点 |
|------|---------|---------|
| **上游** | 原材料/设备/核心零部件 | 供给集中度、是否有瓶颈环节、价格走势 |
| **中游** | 制造/集成/平台 | 技术壁垒、规模效应、毛利率水平 |
| **下游** | 终端应用/渠道/消费者 | 需求弹性、客户集中度、切换成本 |

分析各环节附加值分布和话语权（"微笑曲线"位置）。

### 第四步：竞争格局

基于用户提供的行业材料和AI知识库分析：
- 标注"[请补充行业数据]"提示用户补充

**集中度分析**：
- 当前CR3/CR5/CR10及近3年变化趋势
- 判断格局走向：分散→集中？稳定寡头？还是集中→分散？

**竞争要素矩阵**：

| 玩家 | 市占率 | 核心优势 | 主要劣势 | 战略方向 |
|------|-------|---------|---------|---------|
| [A] | [X%] | | | |
| [B] | [X%] | | | |

**壁垒评估**：
- 技术壁垒：专利数/研发投入/技术代差
- 规模壁垒：产能规模/成本优势/渠道覆盖
- 品牌壁垒：品牌溢价/客户粘性/转换成本
- 政策壁垒：牌照/准入/环保标准

**潜在进入者威胁**：外资进入意愿、跨界玩家、替代技术路线。

### 第五步：驱动因素与风险

**驱动因素评估**：

| 驱动力 | 类型 | 强度 | 持续性 | 说明 |
|--------|------|------|-------|------|
| [因素1] | 需求/供给/政策 | 强/中/弱 | 短期/中期/长期 | [具体说明] |

- 需求端：消费升级/国产替代/新场景/出海
- 供给端：技术突破/产能扩张/成本下降
- 政策端：补贴/标准/准入/环保

**风险因素**：
- 政策风险（政策收紧/补贴退坡/监管趋严）
- 技术替代风险（新技术路线颠覆现有格局）
- 产能过剩风险（集中扩产导致价格战）
- 需求不及预期（渗透率提升放缓/终端需求疲软）
- 原材料价格波动
- 出海/全球化风险（贸易壁垒/地缘政治/汇率波动/合规差异）

### 第六步：投资机会梳理

> 投资标的梳理基于产业链分析推导，仅供研究参考，请结合公司基本面和估值独立判断。

按产业链环节梳理A股投资标的，按受益程度排序。

**标的筛选逻辑**：
1. 业务相关度：主营业务收入中该行业占比>30%
2. 竞争地位：细分环节市占率前3或具备差异化优势
3. 成长确定性：产能/订单/客户验证等可跟踪指标

## 质量标准

- 使用申万行业分类体系
- 市场空间测算须列出公式、假设和数据来源
- 竞争格局须有量化数据（市占率/CR指标）
- 政策梳理须标注发文单位和文号
- 投资标的须标注筛选逻辑和相关度
- 不得使用过时数据而不标注年份
- 产业链分析须覆盖上中下游全链条
- 驱动因素须评估强度和持续性

## 红线规则

1. **不编造市场数据**：市场规模、市占率、增长率等核心数据必须有可溯源出处；未查到的标注"[需验证，建议查阅XX数据源]"
2. **不混淆事实与推断**：行业趋势判断必须区分"已发生的事实""基于数据的合理推演"和"推测"，后两类标注推断依据和信心度
3. **不替代投资决策**：报告中所有投资机会梳理和标的列表仅供研究参考，必须保留风险提示段
4. **引用数据标注时效**：所有数据点必须标注年份/季度，超过12个月的数据提示"数据可能已过时"
5. **政策引用规范**：政策文件引用必须标注发文单位+文号+发布日期，不凭记忆引用政策内容
6. **不越界做定价建议**：不给出"目标价XX元""买入/卖出"等定价或交易建议

## 输入不足处理

| 缺失项 | 处理方式 |
|-------|---------|
| 行业名称模糊（如"新能源"） | 列出可能的细分赛道（光伏/储能/锂电/氢能等），请用户明确具体方向 |
| 未指定研究深度 | 默认执行全景概览模式，在报告开头标注"深度：全景概览（如需深挖请指明环节）" |
| 无可用材料 | 提示用户行业研究依赖数据支撑，建议上传行业白皮书/券商研报/政策文件；无材料时基于公开知识生成框架，关键数据位标注"[待补充]" |
| 侧重方向不明 | 默认覆盖市场空间+竞争格局+产业链，其他环节按标准框架简要覆盖 |
| 行业生命周期不明 | 基于渗透率、增速等可获取数据判断并标注"判断基于XX数据，可能有偏差" |
